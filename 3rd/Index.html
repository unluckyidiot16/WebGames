💡 실제 게임 페이지 연동 방법

이 인덱스 페이지가 정상적으로 해금 기능을 작동하려면, 각 게임(.html)이 끝났을 때(클리어 시) 아래의 자바스크립트 코드를 실행하도록 게임 코드를 수정해야 합니다.

  • 1번 게임 (ABC 팡) 클리어 시: localStorage.setItem('game1_cleared', 'true');
  • 2번 게임 (AtoZ) 클리어 시: localStorage.setItem('game2_cleared', 'true');
  • 3번 게임 (영어단어 스크램블) 클리어 시: localStorage.setItem('game3_cleared', 'true');
  • 4번 게임 (영단어 팡) 클리어 시: localStorage.setItem('game4_cleared', 'true');

※ 주의: 이 인덱스 페이지와 게임 페이지들은 모두 같은 도메인(unluckyidiot16.github.io)에 있어야 진행 상황이 정상적으로 공유됩니다.
